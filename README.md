## “虹烁独立区”是什么？

“虹烁独立区”是一个基于Cutesune¹（曾用名为RyuujinZERO）的Avali种族世界观设定²的二次创作的世界观。该世界观的创作大致遵循“尽量不修改原设定，少量合理延申原设定，在此基础上合理另立新舞台来安置新设定”的原则。

所有关于“虹烁独立区”的内容遵循CC BY-SA的许可协议³。
